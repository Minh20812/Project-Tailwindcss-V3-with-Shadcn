# Project Tailwindcss V3 with Shadcn (WIP: Work In Progress)

## Cài Đặt

1. **Clone repository**
   ```bash
   git clone https://github.com/Minh20812/Project-Tailwindcss-V3-with-Shadcn.git
   ```
2. **Cài đặt các gói cần thiết**
   ```bash
   cd repository-name
   npm install
   ```
3. **Chạy ứng dụng**
   ```bash
   npm run dev
   ```
